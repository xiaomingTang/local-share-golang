同 Wi‑Fi 下文件共享 · 浏览器访问 · 纯绿色

只需一台 Windows 电脑运行，其他设备用浏览器就能共享文件

LocalShare 会在你的电脑上启动一个本地 HTTP 服务：手机、平板、Mac 或其他电脑都无需安装任何客户端， 只要在同一 Wi‑Fi（或同一路由器下的同一网络）里用浏览器打开地址，即可访问、上传、下载、预览与管理文件。

下载最新版（Windows） 快速开始
  • 纯绿色免安装：解压即用，删除文件夹即可卸载
  • 无需管理员权限：默认不修改系统设置
  • 多端零负担：其他设备只要浏览器即可访问（同一 Wi‑Fi）
适用场景
手机和电脑互相分享文件、家庭多设备互传资料、会议室内快速分发/收集文件（同一 Wi‑Fi 内）。

为什么是 LocalShare？

为“快速、便捷地共享文件”而生：不折腾、不要求对方装客户端，打开浏览器就能用。

一台运行，多端访问

只需要在 Windows 电脑上下载并运行。手机、平板、Mac 或其他电脑……都无需安装客户端，浏览器打开地址即可使用。

同一 Wi‑Fi 内，隐私可控

文件共享发生在同一 Wi‑Fi/同一网络内。停止共享即停止服务，不依赖云端中转。

访问口令保护（可选）

可在客户端设置访问口令：打开网页时需先输入口令，通过后会自动记住（临时 token），避免同一 Wi‑Fi 下被随意访问。

权限管理（读/写/删）

可在客户端勾选读/写/删除权限。默认允许读写、禁止删除，让共享更可控。

纯绿色，免安装

下载后解压即可运行，无需安装器、无需系统服务；删除程序目录即可完成卸载。

无需管理员权限

默认无需管理员权限即可运行。可选的 Windows 右键菜单也只写入当前用户范围（HKCU）。

浏览器里管理文件

目录浏览、上传、下载、预览（依赖浏览器支持的文件类型），在同一网络内即可完成。

二维码快速打开

电脑端生成二维码，手机一扫就能打开共享页面，减少手动输入地址的麻烦。

为什么“只需一台电脑运行”很重要？

很多方案的问题不是功能不够，而是使用成本太高：装、配、折腾一圈，分享的兴致先被消耗掉。

传统做法 麻烦 / 挫败
  • 对方也要装客户端/插件：临时分享常被安装/权限卡住
  • 临时分享要配置：步骤多、兼容性不确定
  • 跨设备时更难：手机/Mac 往往“不想装、装不方便”
LocalShare 省心 / 顺滑
  • 只需一台 Windows 电脑下载并运行
  • 其他设备用浏览器打开地址即可
  • 同一 Wi‑Fi 内共享：绿色免安装、无需管理员权限

3 步开始共享

  1. 运行 LocalShare
    下载并解压后双击运行（免安装）。
  2. 选择要共享的文件夹
    点击“选择文件夹并开始共享”，或直接把文件夹拖到窗口。
  3. 其他设备用浏览器访问
    同一 Wi‑Fi/同网段下，打开显示的 URL 或扫描二维码即可。
打不开怎么办？
  • 确认设备在同一 Wi‑Fi（或同一路由器下的同一网络）。
  • 检查 Windows 防火墙/安全软件是否拦截该程序的入站访问。
  • 电脑有多张网卡时，可能会显示非目标网络的 IP，可尝试切换网络后重新开始共享。

Q&A

只需要在一台 Windows 电脑下载并运行吗？其他设备也要装吗？
是的。LocalShare 在 Windows 上启动共享服务；其他设备（手机、Mac 或其他电脑）无需安装客户端，只要在同一 Wi‑Fi/同一网络内用浏览器访问地址即可进行文件浏览、上传、下载和预览。
它是绿色软件吗？需要安装吗？
是纯绿色免安装：解压即用。删除程序文件夹即可卸载。

唯一可能的系统改动来自“可选的 Windows 右键菜单”。不开启就不会写入注册表。

支持“访问口令”吗？怎么用？
支持。你可以在客户端里设置“访问口令”，启用后：访问共享网页会先提示输入口令，通过后会获得一个临时 token（会自动续期），后续操作无需重复输入。

口令规则：1–16 位，仅支持字母/数字。提示：访问口令是局域网内的轻量保护，仍建议在可信网络中使用。

支持“权限管理”吗？
支持。可在客户端设置中勾选读/写/删除权限；默认允许读写、禁止删除。关闭写入后无法上传，关闭读取后无法浏览/下载。
需要管理员权限吗？
不需要。应用本身可直接运行；可选右键菜单写入的是当前用户注册表（HKCU），同样无需管理员权限。
我启用了右键菜单，后来直接删了应用，右键菜单还残留怎么办？

建议：删除应用前先检查一下——如果你曾启用过右键菜单，先在 LocalShare 界面里一键取消右键菜单，再删除应用会更省心。

右键菜单属于注册表项，删除程序文件并不会自动清理。可以按以下方式移除（仅影响当前用户）：

方法 A：PowerShell 一键清理（推荐）

reg.exe delete "HKCU\Software\Classes\Directory\shell\ShareFolder\command" /f
reg.exe delete "HKCU\Software\Classes\Directory\shell\ShareFolder" /f
reg.exe delete "HKCU\Software\Classes\Directory\Background\shell\ShareFolder\command" /f
reg.exe delete "HKCU\Software\Classes\Directory\Background\shell\ShareFolder" /f

# 让资源管理器刷新右键菜单（可选）
Stop-Process -Name explorer -Force
Start-Process explorer

方法 B：手动清理（图形界面）

  1. 按 Win + R，输入 regedit 回车。
  2. 依次定位并删除以下键（如存在）：
    • HKEY_CURRENT_USER\Software\Classes\Directory\shell\ShareFolder
    • HKEY_CURRENT_USER\Software\Classes\Directory\Background\shell\ShareFolder
  3. 重启资源管理器或注销/重启电脑，使右键菜单刷新。

开始共享吧

下载即用，其他设备打开浏览器即可。
